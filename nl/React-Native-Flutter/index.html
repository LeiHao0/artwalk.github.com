Ik praatte met mijn voormalige collega, die een gevorderde frontend was. Hij hielp me met het starten van de updateserver, die ook in Node.js. Ik heb dit project aan hem overgedragen toen ik wegging.

Volgens hem komt dit omdat hij niet weet hoe hij native code moet schrijven, net zoals React React vergelijkbaar is met React.js, waarom zou je het niet gebruiken om zowel iOS als Android te proberen?   • Lelijke codestijl (nemen ze het serieus? Dit is een officieel voorbeeld)

In ieder geval was ik er echt van onder de indruk, het sloeg de ‚Äònative componenten‚Äô over en schreef zijn eigen 2D-rendering-engine. Veel meer getalenteerd dan React Native.
