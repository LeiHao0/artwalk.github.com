Bagaimana mengkonversi video ke teks

Diposting di 2018-07-22 Edited on 2020-12-21 Disqus:
Symbols count in article: 2.4k Reading time ≈ 2 mins.

Jika audionya adalah Mandarin, saya menemukan bahwa XFyun FlySpeechRecognizer melakukan pekerjaan dengan baik dalam hal ini, bahkan dapat mengenali beberapa kata bahasa Inggris sederhana dalam audio bahasa campuran.

Tidak perlu melatih model pembelajaran mesin dari awal untuk melakukan hal seperti itu.

XFyun API memiliki beberapa batasan. Ukurannya harus kurang dari atau sama dengan 2MB, dan panjang audio harus kurang dari atau sama dengan 60-an.

Penggunaan ./test.sh test.mp4

Sekarang kita dapat menggunakan XFyun untuk mengenali audio.

Ini adalah dokumen mereka XFyun IFlySpeechRecognizer, Tetapi kode sampel tidak dapat dijalankan, jadi saya melakukan beberapa perbaikan.
