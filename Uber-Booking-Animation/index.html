Uber booking animation

Posted on 2019-10-03 Edited on 2020-12-21 Disqus:
Symbols count in article: 4.7k Reading time ≈ 4 mins.

In June of this year, I joined WWDC in San Jose. It was the waiting animation I saw when I met my friends at the Cupertino Library using Uber.

The animation is awesome!

I was curious how they did it, so I did some exploration.

Preview

Well, the polyline animation is still not smooth enough. Does anyone have a better idea?

Step by step

Rotate the map

I chose Map as the default map to demonstrate for the following two reasons:

  • After WWDC19, Simulator is built on the Meta and Core layers of all Apple frameworks included in MapKit, and Metal uses the GPU of the underlying Mac system.
  • Google Map requires a developer key to access, which means that after forking the repository from GitHub, you cannot run it directly on the emulator

 Map transactions can be adjusted by embedding in the UIView.animate block

Google Map must use CATransaction

Fixed animation

We need two CAShapeLayer to make the animation, one for the circle and the other for the pushpin.
Note that the circle disappears after reaching the maximum value, and the pin has an easyIn curve.

Animated polyline
