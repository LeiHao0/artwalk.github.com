Use UIStoryboardSegue to customize the presentation transition of UIViewController

Posted on 2016-12-20 Edited on 2020-12-21 Disqus: Symbols count in article: 1.5k Reading time ≈ 1 mins.

The most popular way is UIViewControllerAnimatedTransitioning

In order to do this, you must:

  • Create a new subclass, which can be called “XXXTransitioning”, inherited from “NSObject”, and implement the two functions of the protocol “UIViewControllerAnimatedTransitioning”.
  • Then:
    1. Create a new instance of “XXXTransitioning” in “ViewController”.
    2. Set transitioningDelegate = self.
    3. Use UIViewControllerTransitioningDelegate to extend “ViewController” and implement it.
