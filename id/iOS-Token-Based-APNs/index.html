APN berdasarkan token iOS

Diposting di 2020-05-29 Edited on 2020-12-21 Disqus:
Symbols count in article: 481 Reading time ≈ 1 mins.

Apple menyediakan metode baru (token otentikasi) untuk menjalankan APN

Dibandingkan dengan metode sebelumnya (sertifikat penyedia), sertifikat * .p12 akan kedaluwarsa setiap tahun

Berbasis token * .p8 ** Tidak pernah kedaluwarsa **

  • Buat koneksi berbasis token dengan APN
  • Komunikasi dengan APN

Untuk pelanggan

Satu-satunya perbedaan pada klien adalah

  1. Buat kunci penandatanganan token otentikasi di https://developer.apple.com/
  2. Unduh * .p8 dan kirimkan ke teknisi backend

Untuk bagian belakang

Banyak kerangka kerja yang mendukung * .p8.
