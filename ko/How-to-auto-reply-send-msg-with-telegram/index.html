Telethon

  • pyenv를 사용하여 파이썬 버전을 전환하려고했지만 작동하지 않았습니다.

  • 마지막으로 여기에서 [macOS 10.15에서 Python 충돌](https://stackoverflow.com/questions/58272830/python-crashing-on-macos-10-15-beta-19a582a-with-usr-lib-libcrypto- dylib),OpenSSL이 없습니다.
