To the dead 2012

Diposting di 2013-03-27 Edited on 2020-12-21 Disqus:
Symbols count in article: 6.5k Reading time ≈ 6 mins.

Saya masih ingat. Awalnya ringkasan Hari Tahun Baru. Ditunda hingga Festival Musim Semi, lalu hari ini.

Ditulis pada 2012

Saya gelisah sebulan sebelum Festival Musim Semi
Saya tidak tahu apakah itu ada hubungannya dengan orang di luar
Ketika saya pulang saat Tahun Baru Imlek, saya melihat salju beterbangan di langit dan bumi tertutup perak di bus, dan hati saya kembali damai.

2012 adalah tahun saya tumbuh paling pesat
Mengesampingkan perubahan di lingkungan eksternal, kita tidak dapat lagi menggunakan standar siswa untuk menuntut diri kita sendiri

Pada paruh pertama tahun ini, saya tidak punya pekerjaan, membaca buku dan bermain game di rumah
Ingat karya dewa â€œTake Time as a Friendâ€
Hati sangat bersemangat, dan saya membacanya sesekali selama sebulan
Setelah itu, dunia terasa berbeda
Mungkin sejak itu, jatuh cinta pada membaca

Pembunuhan berlebihan

Saya masih ingat proses Tai Chi dari kaku ke lembut di tahun kedua saya. Saya pernah mengobrol dengan guru saya sekali
Dia berkata bahwa saya gemetar terlalu keras, dan meminta teman sekelas saya untuk merekam video dengan ponsel mereka, dan saya akan mengerti dengan melihat mereka.
Pada saat itu, saya terpana dan berkata, saya pikir ini satu-satunya cara untuk pergi.

Ya, ini berlebihan

Saya juga merasakan sakit punggung ketika saya masih di sekolah menengah
Saya telah terkena pengobatan tradisional sejak saat itu
Tapi belakangan ini saya merasakan sakit di anggota tubuh dan bantal leher saya
Saya mendengar sebuah kalimat dalam Teori Relativitas Dongwu, â€œKetika Anda menyadari keberadaannya, itu mungkin menjadi masalah.â€
Saya pikir itu lebih serius dan bertanya-tanya apakah saya akan mati
Tapi baru-baru ini saya melihatnya di â€œThe Art of Learningâ€, itu mungkin juga pengantar Tai Chi, dan saya merasakan sesuatu yang tidak saya rasakan sebelumnya
Singkatnya, semuanya berjalan dan tetap tenang

Sekilas

Terkadang sangat aneh, Anda bisa melihatnya sekilas di kerumunan
Meskipun ini bukan pandangan pertama, setelah beberapa saat, dalam kerumunan besar, Anda akan melihat orang ini
Hanya dengan melihatnya dapat membedakannya dari orang-orang di sekitarnya
Latar belakang, kepribadian, pekerjaan, hobi semuanya tidak diketahui, tetapi Anda masih bisa merasakan istimewanya

Pergi ke Westside Ice Rink pada hari Sabtu, ini adalah kedua kalinya dalam hidup saya skating
Seperti skating
Salah satunya adalah membuat diri Anda sendiri merasakan kesulitan menjadi hewan yang tegak lagi dan melatih otak kecil Anda
Yang kedua adalah memahami kembali keseimbangan, ini dan Tai Chi harus memiliki efek yang saling melengkapi
Jadi putuskan untuk menjadikannya konstan untuk olahraga akhir pekan
Saat ini, kemudinya masih menggunakan gesekan, saya berharap bisa menggunakan gaya sentripetal di lain waktu

terburu nafsu

Setiap orang mengatakan bahwa ini adalah masyarakat yang terburu nafsu

Ya, tren umum
Ada pepatah: Jika Anda tinggal di tempat yang tidak normal untuk waktu yang lama, Anda akan menganggap kelainan itu dibenarkan.

Saya tidak menyangkal bahwa saya terburu nafsu
Saya ingin melakukan segalanya, saya tidak terlibat apa-apa
Tapi saya merasa tidak berdaya, membaca dan memperhatikan hanya dimulai dari tahun senior.
Mengetahui GTD, apa gunanya
Kuncinya adalah kualitas penggunaan waktu, bukan kuantitas
aliran- aliran jantung, waktu berhenti untuk Anda

Hal yang sama berlaku untuk ketegasan, saya merasa seperti bagaimana menangani hubungan yang dalam dan luas
Pertama-tama, seperti yang dikatakan Qiao, teruslah mencoba untuk menemukan, dan akhirnya, Anda akan menemukan minat Anda
Saya telah bergumul dengan pemrograman selama dua tahun di universitas. Saya secara naif mengira itu adalah hal seumur hidup.
Dan berpikir bahwa selama Anda disebut master atau orang hebat, Anda tidak perlu khawatir tentang uang
Dan mengaku bekerja demi bunga, bukan uang
Lihat sekarang, itu sangat konyol dan naif
Dalam empat tahun kuliah yang tanpa beban, saya tidak berhasil melebar. Meskipun sekarang agak terlambat, saya masih harus terus melakukannya.

Bergerak

Terima kasih kepada teman-teman di Hefei (saya ternyata muak dengan istilah ini, tetapi kemudian menemukan bahwa ada arti lain, budaya Tionghoa sangat luas dan mendalam), saya biasa menarik Anda keluar setiap kali saya sedih.
Dan tinggal bersamaku sebentar
Siapa lagi, ketika saya mengetahui bahwa saya memiliki lima mata pelajaran dan mengambil kembali satu, tidak mendapatkan gelar sarjana di bidang psikologi, tidak masuk sekolah pascasarjana, putus asa untuk gaji yang rendah dan terburu-buru, dan mempersiapkan sehari sebelum ulang tahun saya Saat Anda dipukul, sedih, dan kacau, tidak ada rasa jijik selain kenyamanan

Sanshengshi

Mendengarkan namanya saja sudah luar biasa

Kontak pertama ada di â€œSou Shen Jiâ€:

Berbaring di atas batu tiga kehidupan, sadari kehidupan yang lalu dan yang akan datang, minumlah mata air tanpa rasa khawatir, dan lupakan dendam

Terakhir kali saya mengunjungi Hangzhou dan mencari tempat wisata di Hangzhou di Google, saya menemukan tiga kata â€œSanshengshiâ€

Di penghujung usia saya yang berusia 22 tahun, banyak hal yang melenceng dari penglihatan saya
Dan saya menemukan bahwa semua yang saya lakukan mungkin menghindari (apa yang harus dihindari?)

Hal yang paling menyedihkan adalah setelah tidur, otak menyusun ulang dan menjalankan segala sesuatunya sesuai dengan keinginannya.

Ketika saya bangun, saya mendapati diri saya tidur sangat nyenyak dan santai, seperti saat saya mendengar Zhao memberi tahu saya bahwa saya menderita BL di tahun pertama saya.

Lalu aku pergi menemui sanshengshi, bukan untuk yang lain, lagipula, itu namaku untuk menghapus kehidupan.

Utilitas dan Kebebasan

Pahami liberalisme sejati, pahami peran pajak warisan,

Saya mengerti, saya mungkin tidak memiliki pandangan independen, dan ada kesenjangan besar antara berpikir kritis dan berpikir kritis.

Saya memahami bahwa sejak â€œKelas Kebahagiaanâ€, saya merasa sangat baik tentang diri saya sendiri, dan bahkan terlalu dikoreksi

Pahami bahwa Anda masih tidak bisa membuat janji kepada orang lain, tidak peduli siapa orang itu

Saya mengerti bahwa agar tidak mengecewakan diri sendiri atau orang lain, saya sering memilih untuk tidak bertindak

Pahami bahwa terkadang peluang digunakan untuk menunggu dan berubah, hasil akhirnya hilang

Saya mengerti bahwa begitu saya merasa frustrasi, saya masih merasa sedih

Tapi saya masih belum mengerti, bagaimana kapan dimana siapa kenapa bagaimana

Terkadang saya merasa bahwa semua yang saya lakukan hanyalah kekacauan kecil, jadi apa jadinya jika saya mengubahnya, saya masih merasakan ketidakberdayaan

Kayu pinus bawah Yu Yujian, bibit dari gunung
Dalam diameter batang, teduh seratus kaki ini

Wan berkata dia baru-baru ini menemukan bahwa jika Anda bekerja keras, Anda akan mencapainya

Tetapi model saya mungkin tidak terealisasi, setidaknya akan ada banyak celah antara hasil nyata dan pemodelan saya sendiri

berharap

Tentu saja, saya pikir, mungkin mereka hanya mencoba untuk membenarkan diri mereka sendiri dan menemukan alasan yang terdengar bagus untuk diri mereka sendiri
Tentu saja, lebih mungkin, apa yang saya katakan salah
Tetapi bagaimanapun juga, tidak peduli bagaimana Anda mendesah, 22 yang indah telah berlalu.
Terkadang terasa seperti ada jam yang berdetak di jam alarm
Apakah itu pengamatan waktu batin, tetapi juga pengalaman perjalanan waktu
Memang benar, orang mati seperti suami

Saya juga tahu bahwa harapan tidak masalah apakah ada atau tidak sama sekali
Tapi saya tidak bisa tidak berharap bahwa saya sedang dalam perjalanan menuju akhir siklus ketiga.
Saya bisa membuat terobosan, seperti nama game Microsoft: The Rise of the Country
