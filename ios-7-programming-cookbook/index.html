How to read the cookbook

Oh my God, it’s finally over, more than 1k pages! ! !
Hand-typed 80% of the code inside and found two bugs in the code
In addition, if there is an error, children’s shoes can Pull Requests on Github to oreillymedia/iOS7_Programming_Cookbook

Suitable for the crowd:

Generally speaking, the Cookbook series are analogous to dictionaries. There is no need to go through the top and bottom of what is used and what to look up.

But after brushing it again, the benefits of building the relevant knowledge system are obvious

Not suitable:

  1. noob
  2. Senior iOS Developer

Novices can read these two books first:
Head First iPhone and iPad Development
The Big Nerd Ranch Guide

method:

  1. First of all, be prepared for collapse →_→
    Think about how many children’s shoes back to the abandon’s dictionary and kneel down

  2. Select the chapters that are key, that is, the chapters related to your current work look at it without missing a word
    Since the stuff you see can be used immediately, the effect is better
    The non-code parts are the principles and tips, so take a closer look

  3. When it reaches about 50%, the focus turns to the code
    At this time, I’m basically familiar with the author’s narrative method. Scan the title and you basically know what to do. Combine the code and guess with a guess. Just scan the text.

  4. Know how to give up
    Memorizing the dictionary does not allow you to speak fluent English, so there is no need to finish this book.
    For example, when I watched it, I thought iCloud and the like were too troublesome. If you simply synchronize data, it’s better to set up a backend directly, so just skip it.
    It’s just in line with the 28th rule.

  5. Pairing
    The effect of working alone is not as high as group learning
    But what you can see is different. For example, my friends are watching the Python Cookbook. It’s enough to achieve mutual promotion.
    PS: When I watched more than half of it, I realized that he was watching the translated version…

  6. English
    iOS update is too fast, the latest ones are all in English, 7 has not been translated yet, the official version of 8 will be released…
    So, learn English well, Sao Nian
