Cara mengunduh video dari Toutube secara bersamaan dan menyematkan subtitle secara berkelompok

Diposting di 2018-08-10 Edited on 2020-12-21 Disqus:
Symbols count in article: 1.6k Reading time ≈ 1 mins.

Yang harus saya lakukan adalah mengunduh beberapa kursus, yang sangat membantu saya di youtube, dan menyematkan subtitle Zh-Hans dalam video, dan kemudian mengirimkannya ke situs web berbahasa Mandarin yang tidak mendukung subtitle eksternal.

Alat ini dapat membantu siswa China yang tidak berbicara bahasa Inggris dengan baik dan tidak dapat mengakses youtube.

Konfigurasi dl Youtube

Path file adalah ~ / .config / youtube-dl / config

Download secara bersamaan

Nama file terjemahan batch

Setelah mengunduh semua video dalam daftar, gunakan

ls | grep mkv | sed "s / \. mkv //"

Kemudian salin dan tempel ke Google untuk mendapatkan teks terjemahan, lalu simpan ke zh.txt

Sematkan subtitle secara berkelompok

Pada Intel Core i7, saya biasanya menghabiskan sepanjang malam merekam 20 video, dan satu jam video membutuhkan waktu 20 menit untuk menyematkan subtitle-nya.
