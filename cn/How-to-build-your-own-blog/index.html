建立自己的博客

发表于 2014-01-11 更新于 2023-08-07 Disqus：
本文字数： 747 阅读时长 ≈ 1 分钟

个人感觉最好的是：

VPS+域名 WordPress 等博客框架，这样网址，后台环境什么的都是由自己来控制的

要是不花钱，只能寄人篱下了

比如挂到 GAE 或者 SAE 下

还有一种最简单的是挂到 Github 上，
大致过程是

  1. Github 上新建一个 Repository，xxx.github.com
  2. Jekyll 或者 Hexo 等静态博客生成工具
  3. 没有了

对，就这么简单，并且网上教程一片片，就不重复了

我用的是 Hexo，挂在这里

相对 Jekyll，感觉更简单些

说点不相干的：

还记得小学时，天天让写作文，那时真是被虐的体无完肤

每天上学放学写（chao）作业，哪来的感悟，还一天一篇，现在最多一周才能有点东西可以记录好吧

怎么办呢，只能抄范文喽

可惜那时没有理论指导，抄的不精，一本什么 小学生范文大全抄完之后，抄无可抄，无奈打起了语文课本的主意

太岁头上动土，不是早死么，结果自然是被发现喽

然后那被誉为人类灵魂工程师的三年级语文教师，拿着我的日记本当着全班同学的面狠狠地扇了我脑袋

哦（wocao），杀鸡儆猴也不用这么狠吧

当时我隐忍不发，默默提升技能，毕竟高考还是有八股文存在的，为了明天，小不忍则乱大谋（好吧，当时我很傻的，感觉老师打学生挺正常的，就像前些年还认为爸爸打儿子那样）

还好大学了终于不用写那些恶心的东西了

大一时有个室友鄙视我空间好烂，然后去装扮，挂了 N 多的 Flash，卡的人家都打不开

后来发现有日志功能，随便写了点关于学校的事情，然后有一些同学评论，蛮开心的，然后就写着了

一年多后我才知道这东西叫 博客

后来编程方面的东西都挂到 CSDN 去了，看着那左下角的千里之外，默默离开了

快毕业时有次搜索东西，看到一丫写的东西好熟悉，看了下访问量 1W 多，居然还有排名，心想这是谁啊，仔细一看，居然是我…

再后来就是想找个没人认识的地方，记录点东西
