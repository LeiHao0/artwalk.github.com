Jika kernel_task menggunakan sebagian besar CPU Mac Anda

Saya telah memperhatikan sebelumnya bahwa kernel_task akan 100% macet di setiap kesempatan, tetapi ini adalah prosesSistem, tidak mungkin

根据  的 文档 Jika kernel_task menggunakan sebagian besar CPU Mac Anda

Dengan kata lain, kernel_task merespons kondisi yang menyebabkan ** CPU Anda menjadi terlalu panas **, meskipun Mac tidak terasa panas bagi Anda. Itu sendiri tidak menyebabkan kondisi tersebut. Saat suhu CPU menurun, kernel_task secara otomatis mengurangi aktivitasnya.

Ini karena CPU terlalu panas 🔥 meskipun menurut Anda tidak panas 🥶 …

Selain itu, kernel_task bukanlah pelakunya. Saat CPU tidak panas, tingkat huniannya otomatis akan turun …

Saya juga menemukan bahwa itu tidak boleh di sebelah kiri 👈 pengisian 🔋, percayalah

[Inilah alasan mengapa Anda tidak perlu mengisi daya MacBook Pro di sisi kiri](https://www.digitaltrends.com/computing/why-you-shouldnt-charge-macbook-pro-left-side/#:~:text = Sementara% 20charging% 20the% 20MacBook% 20Pro, gunakan% 20the% 20right% 2Dhand% 20ports.)

Saat mengisi daya MacBook Pro dan memasang periferal ke port sebelah kiri, port perangkat bisa menjadi panas; untuk beberapa alasan, ini dapat mengakibatkan kernel_task menjadi sedikit rusak dan menggunakan banyak muatan sumber daya sistem.

Tetapi setelah beralih ke 👉 pengisian 🔋 kanan setelah beberapa hari, itu 100% setelah beberapa saat

putus asa…

Tiba-tiba bertanya-tanya apakah itu karena kebutuhan untuk membersihkan

Cara Membersihkan Debu & Kipas MacBook

Google baru tahu bahwa Anda dapat membongkarnya sendiri
Punyaku adalah buku berusia 16 tahun, yang dijamin
不然 根据 [Garansi Mac](https://www.apple.com/legal/war Warranty/products/embedded-mac-war Warranty-rest-of-apac-english.html)
Jika Anda membongkar sendiri, Anda akan kehilangannya

Garansi ini tidak berlaku:

  • (c) kerusakan yang disebabkan oleh penggunaan komponen pihak ketiga atau produk yang tidak memenuhi spesifikasi Produk Apple

Alat 🔧

  • Sekrup penta 🔩 pisau
  • Bola jelaga, sikat kecil
  • Kartu kredit terbengkalai 💳

langkah

  1. Melepaskan
  2. Buat celah kecil di samping
  3. Masukkan kartu kredit yang tidak terpakai 💳 dan geser searah panah biru
    1. Anda akan mendengar bunyi klik
    2. Pada Gambar 2 Anda dapat melihat bahwa ada empat gesper di garis horizontal tengah
    3. Daripada merusak sesuatu (saya harap begitu 😓)
  4. Gunakan secara simetris di kanan 💳 untuk meluncur
  5. Kemudian tekan panah merah untuk menariknya secara horizontal
    1. Gambar 2 Slot kartu dapat dilihat di atas kipas
    2. Jadi paksa secara horizontal, tarik ke kiri dan kanan ↔️ sedikit secara simetris
  6. Penutup belakang dibuka

Sangat kotor

Setelah membersihkan debu, kernel_task tidak banyak muncul, dan 💻 juga dihidupkan kembali
Saya langsung merasakan bahwa notebook selama 16 tahun 💻 dapat digunakan selama 5 tahun 👨‍💻
