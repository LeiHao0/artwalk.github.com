總算刷完了

怪不得說 Ruby 給你自由，單單從語法糖方面來看，做的比 Python 好
之前還因為 ‘end’ 問題，不喜歡 Ruby，但整體做下來：

  1. 不用像 Python 那樣處處注意縮進，還能自動縮進(Python 就沒見過)，省心
  2. ：等符號減少，函數調用更自然
  3. 更加面向對象 (python 還是有 類型(int str …) 的概念, 傳參，返回值等特別明顯；Ruby 弱化了，用的時候甚至感覺不到類型的存在)

整體下來，也就是實現了

  1. 與算法無關代碼，比 Python 又少了許多
  2. 寫起來更加自然

所以，在沒了解一個東西前，最好不要帶主觀判斷

PS：
Java 和 Android 用了將近一年，就是無愛，這個不算不了解吧
