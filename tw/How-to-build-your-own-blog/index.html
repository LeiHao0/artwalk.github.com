建立自己的博客

Posted on 2014-01-11 Edited on 2020-12-21 Disqus:
Symbols count in article: 747 Reading time ≈ 1 mins.

個人感覺最好的是：
VPS+域名 WordPress 等博客框架，這樣網址，後台環境什麼的都是由自己來控制的

要是不花錢，只能寄人籬下了
比如掛到 GAE 或者 SAE 下

還有一種最簡單的是掛到 Github 上，
大致過程是

  1. Github 上新建一個 Repository，xxx.github.com
  2. Jekyll 或者 Hexo 等靜態博客生成工具
  3. 沒有了

對，就這麼簡單，並且網上教程一片片，就不重複了
我用的是 Hexo，掛在這裡
相對 Jekyll，感覺更簡單些

說點不相干的：

還記得小學時，天天讓寫作文，那時真是被虐的體無完膚
每天上學放學寫(chao)作業，哪來的感悟，還一天一篇，現在最多一周才能有點東西可以記錄好吧
怎麼辦呢，只能抄範文嘍

可惜那時沒有理論指導，抄的不精，一本什麼 小學生範文大全抄完之後，抄無可抄，無奈打起了語文課本的主意
太歲頭上動土，不是早死麼，結果自然是被發現嘍

然後那被譽為人類靈魂工程師的三年級語文教師，拿著我的日記本當著全班同學的面狠狠地扇了我腦袋

哦(wocao)，殺雞儆猴也不用這麼狠吧

當時我隱忍不發，默默提陞技能，畢竟高考還是有八股文存在的，為了明天，小不忍則亂大謀(好吧，當時我很傻的，感覺老師打學生挺正常的，就像前些年還認為爸爸打兒子那樣)

還好大學了終於不用寫那些噁心的東西了
大一時有個室友鄙視我空間好爛，然後去裝扮，掛了 N 多的 Flash，卡的人家都打不開
後來發現有日誌功能，隨便寫了點關於學校的事情，然後有一些同學評論，蠻開心的，然後就寫著了
一年多後我才知道這東西叫 博客

後來編程方面的東西都掛到 CSDN 去了，看著那左下角的千里之外，默默離開了

快畢業時有次搜索東西，看到一丫寫的東西好熟悉，看了下訪問量 1W 多，居然還有排名，心想這是誰啊，仔細一看，居然是我…

再後來就是想找個沒人認識的地方，記錄點東西
