Recuerde la ecuación de distribución gaussiana

$$g(x)=\frac{1}{\sigma \sqrt{2\pi }}^{-\frac{1}{2}( \frac{(x-\mu )}{\sigma} )^{2}}$$

Dado que el monte Fuji 🗻 no es un elemento bidimensional, no se puede utilizar directamente. Debes deformar la fórmula y eliminar $ \ mu $

$$z=\frac{20}{\sigma \sqrt{2\pi }}^{-\frac{R^{2}}{2\sigma^{2}} }, \sigma= 2, R = \sqrt{ x^{2} + y^{2}}$$

Por supuesto, la altura y la pendiente son incorrectas.
Puedes usar la regresión lineal para probarlo. Naturalmente, no quieres un tono humano.
Por supuesto, la parte superior es cóncava y se debe superponer una curva de sonrisa.
