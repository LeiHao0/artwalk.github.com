Posted on 2018-07-22 Edited on 2020-09-09 Disqus:

I found XFyun FlySpeechRecognizer do a quite good job on this thing if the audio is mandarin Chinese, it even can recognize some simple English words in hybrid language audio.

There is no need to train a Machine Learning model from the very beginning to do such things.

XFyun API has some limitations that the size should <= 2MB, audio length <= the 60s.

Usage ./test.sh test.mp4

Now we can use XFyun to recognize audio.
