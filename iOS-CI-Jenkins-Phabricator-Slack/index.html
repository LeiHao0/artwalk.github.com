ResourceRules.plist

On the first day, it was prompted that ResourceRules.plist was wrong, so this was added to the project and Jenkins configuration
Custom xcodebuild arguments $(SDKROOT)/ResourceRules.plist

As a result, it was prompted that ResourceRules.plist could not be found the next day. It was OK to delete this in Jenkins. It’s amazing.
